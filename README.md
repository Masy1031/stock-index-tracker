# Stock Index Tracker (株価指数トラッカー)

## 概要
S&P500やオールカントリー（全世界株式）といった主要な株価指数の動きを、自動で取得・表示するWebアプリケーションです。
投資信託や株式投資に興味がある方が、日々の市場動向をひと目で確認できるように設計されています。

## 特徴
このアプリには以下の機能があります：

1.  **見やすいダッシュボード**
    *   S&P500とオールカントリーの現在の価格と、最終更新日を並べて表示します。
2.  **インタラクティブなチャート**
    *   過去の株価推移をグラフで表示します。マウスを合わせると、その日の具体的な価格を確認できます。
3.  **詳細データテーブル**
    *   「Daily（日次）」ボタンで毎日ごとの細かい値動きを、「Monthly（月次）」ボタンで月ごとの大きなトレンドを切り替えて確認できます。
    *   データが多い場合はページ送りで快適に閲覧できます。

## 使い方（はじめかた）

このアプリケーションをお手元のパソコンで動かすための手順です。
※事前に `Node.js` というソフトがインストールされている必要があります。

### 1. 準備（インストール）
ターミナル（コマンドプロンプト）を開き、以下のコマンドを入力して必要なプログラムをダウンロードします。

```bash
npm install
```

### 2. 起動
準備ができたら、以下のコマンドでアプリを起動します。

```bash
npm start
```

コマンドを実行すると、ブラウザが自動的に開き（または `http://localhost:5173` にアクセスすることで）、アプリが表示されます。

## 技術的なこと（エンジニア向け情報）
- **データ取得**: `yahoo-finance2` ライブラリを使用して、信頼性の高い市場データを取得しています。
- **画面表示**: `React` と `Tailwind CSS` を使用し、モダンで見やすいデザインを構築しています。
- **グラフ描画**: `Recharts` を採用し、美しく動作の軽いチャートを実現しました。
